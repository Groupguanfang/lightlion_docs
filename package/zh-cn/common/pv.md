# 全站PV

* 请求方式：`GET`
* URL: `/pv`

## 成功返回示例

data即为网站pv。

```json
{
  "code": 200,
  "message": "成功",
  "id": 3,
  "meta": "siteView",
  "data": "2"
}
```
